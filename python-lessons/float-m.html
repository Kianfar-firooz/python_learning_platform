🔢 متدهای اعشاری در پایتون

اعداد اعشاری (Float) یکی از دیتا تایپ‌های پایتون برای ذخیره اعداد با اعشار هستن. در این درس با متدهای اعداد اعشاری آشنا میشیم.

📌 عدد اعشاری چیه؟

عدد اعشاری یک عدد با ممیز شناور (floating-point) هست که با . مشخص میشه. توی پایتون با float شناخته میشه.

# تعریف اعداد اعشاری pi = 3.14 science = 1.5e10 # ۱.۵ × ۱۰¹⁰ negative = -2.5

🧰 متدهای عدد اعشاری

.as_integer_ratio()
عدد را به صورت کسر (صورت، مخرج) برمی‌گرداند
(0.5).as_integer_ratio() → (1, 2)
.is_integer()
بررسی می‌کند که عدد صحیح است یا خیر
(5.0).is_integer() → True
.hex()
نمایش هگزادسیمال عدد را برمی‌گرداند
(1024.5).hex() → '0x1.002p+10'
.fromhex(string)
تبدیل هگزادسیمال به float (متد کلاس)
float.fromhex('0x1.ffffp10') → 2047.984375
.from_number(number)
تبدیل عدد به float (پایتون ۳.۱۴+)
float.from_number(5) → 5.0
.real (صفت)
بخش حقیقی عدد را برمی‌گرداند
(3.14).real → 3.14
.imag (صفت)
بخش موهومی عدد را برمی‌گرداند (همیشه ۰)
(3.14).imag → 0.0

🎯 مثال عملی: محاسبات اعشاری

# کار با اعداد اعشاری x = 3.14 y = 2.0 # بررسی صحیح بودن print(y.is_integer()) # True # تبدیل به کسر print(x.as_integer_ratio()) # (157, 50) # بخش‌های عدد print(x.real) # 3.14 print(x.imag) # 0.0
🏋️ تمرین:

اعداد زیر را بررسی کنید:
a = 7.0
b = 3.5
با استفاده از متدهای float مشخص کنید:
۱. کدام یک عدد صحیح است؟
۲. کسر معادل هر کدام چیست؟
۳. نمایش هگزادسیمال a را پیدا کنید
