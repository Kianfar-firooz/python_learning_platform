📦 متدهای تاپل در پایتون

تاپل (Tuple) یکی از دیتا تایپ‌های پایتونه که شبیه لیسته، اما تغییرناپذیر (immutable) هست. در این درس با متدهای تاپل آشنا میشیم.

📌 تاپل چیه؟

تاپل یک دنباله تغییرناپذیر از اشیاءه که با ( ) تعریف میشه. تاپل‌ها سریع‌تر از لیست‌ها هستن و برای داده‌هایی که نباید تغییر کنن مناسب‌ترن.

# تعریف یک تاپل my_tuple = (1, "سلام", 3.14) print(my_tuple[0]) # خروجی: 1 # تاپل یک عضوی (به کاما توجه کنید!) single = (5,)

🧰 متدهای تاپل

.count(x)
تعداد دفعات تکرار x را برمی‌گرداند
(1, 2, 2, 3).count(2) → 2
.index(x, start, end)
ایندکس اولین رخداد x را برمی‌گرداند
(1, 2, 3).index(2) → 1

🎯 مثال عملی: ذخیره اطلاعات ثابت

# اطلاعات یک دانشجو (تغییرناپذیر) student = ("علی", 20, "علوم کامپیوتر") # دریافت اطلاعات name, age, major = student # unpacking # شمارش تعداد یک مقدار print(student.count("علی")) # 1 # پیدا کردن ایندکس print(student.index(20)) # 1
🏋️ تمرین:

یک تاپل از ۵ عدد مختلف بسازید.
سپس:
۱. تعداد دفعات تکرار یک عدد خاص را بشمارید
۲. ایندکس یک عدد دیگر را پیدا کنید
