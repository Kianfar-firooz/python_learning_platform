🔢 متدهای اعداد صحیح در پایتون

اعداد صحیح (Integer) یکی از پایه‌ای‌ترین دیتا تایپ‌های پایتون برای ذخیره اعداد بدون اعشار هستن. در این درس با همه متدهای اعداد صحیح آشنا میشیم.

📌 عدد صحیح چیه؟

عدد صحیح یک عدد کامل (بدون ممیز) هست که می‌تونه مثبت، منفی یا صفر باشه. در پایتون با int شناخته میشه و محدودیت اندازه نداره!

# تعریف اعداد صحیح age = 25 temperature = -5 big_number = 12345678901234567890 # بدون محدودیت! print(type(age)) # <class 'int'>

🧰 همه متدهای اعداد صحیح

.as_integer_ratio()
عدد را به صورت کسر (صورت، مخرج) برمی‌گرداند
(5).as_integer_ratio() → (5, 1)
.bit_count()
تعداد بیت‌های ۱ را در نمایش دودویی برمی‌گرداند
(5).bit_count() → 2
.bit_length()
تعداد بیت‌های مورد نیاز برای نمایش عدد در دودویی
(5).bit_length() → 3
.to_bytes(length, byteorder, *, signed=False)
عدد را به آرایه‌ای از بایت‌ها تبدیل می‌کند
(1024).to_bytes(2, 'big') → b'\x04\x00'
.from_bytes(bytes, byteorder, *, signed=False)
آرایه بایت را به عدد صحیح تبدیل می‌کند (متد کلاس)
int.from_bytes(b'\x04\x00', 'big') → 1024
.real (صفت)
بخش حقیقی عدد را برمی‌گرداند (خود عدد)
(5).real → 5
.imag (صفت)
بخش موهومی عدد را برمی‌گرداند (همیشه ۰)
(5).imag → 0

📊 سیستم‌های شمارش در پایتون

# نمایش اعداد در مبناهای مختلف print(0b1010) # دودویی (binary) → 10 print(0o12) # هشتی (octal) → 10 print(0xA) # هگزادسیمال (hex) → 10 # تبدیل به رشته در مبناهای مختلف print(bin(10)) # '0b1010' print(oct(10)) # '0o12' print(hex(10)) # '0xa'

🎯 مثال عملی: عملیات بیتی و تبدیلات

# کار با بیت‌ها num = 42 # 101010 در دودویی # تعداد بیت‌های ۱ print(num.bit_count()) # 3 # تعداد بیت‌های مورد نیاز print(num.bit_length()) # 6 # تبدیل به بایت bytes_data = num.to_bytes(1, 'big') print(bytes_data) # b'*' # تبدیل از بایت print(int.from_bytes(b'*', 'big')) # 42
🏋️ تمرین:

عدد 255 را در نظر بگیرید.
با استفاده از متدهای int:
۱. تعداد بیت‌های ۱ را پیدا کنید
۲. تعداد بیت‌های مورد نیاز برای نمایش را پیدا کنید
۳. عدد را به آرایه ۲ بایتی تبدیل کنید
۴. کسر معادل آن را پیدا کنید
