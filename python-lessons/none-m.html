❌ متدهای NoneType در پایتون

None یکی از خاص‌ترین دیتا تایپ‌های پایتونه که نشان‌دهنده "هیچ مقداری" یا "تهی" هست.

📌 None چیه؟

None یک شیء از نوع NoneType هست که نشان‌دهنده عدم وجود مقدار است. در پایتون، None با حرف بزرگ نوشته میشه و معادل null یا nil در زبان‌های دیگهست.

# تعریف None nothing = None print(nothing) # None print(type(nothing)) # <class 'NoneType'>

🧰 متدهای NoneType

❌ نوع داده NoneType هیچ متدی ندارد!

💡 ولی: می‌تونید از is برای مقایسه با None استفاده کنید:
x is None → True
x is not None → False

📊 کجاها از None استفاده میشه؟

مقدار پیش‌فرض
برای متغیرهایی که مقدار اولیه ندارن
result = None
بازگشت از توابع
توابعی که مقداری برنمی‌گردونن
print(print("hi")) → None
مقدار پیش‌فرض آرگومان
برای آرگومان‌های اختیاری
def func(x=None):
حذف یک مقدار
برای پاک کردن یک متغیر
del x (متغیر حذف میشه)

🎯 مثال عملی: استفاده از None

# تابعی که ممکن است مقداری برنگرداند def find_user(user_id): if user_id == 1: return "علی" return None # کاربر پیدا نشد # استفاده از تابع user = find_user(2) if user is None: print("کاربر وجود ندارد!") else: print(f"کاربر: {user}")
🏋️ تمرین:

یک تابع بنویسید که یک لیست را دریافت کند و اگر لیست خالی بود، None برگرداند، در غیر این صورت اولین عضو لیست را برگرداند.
سپس تابع را با لیست‌های مختلف تست کنید.
