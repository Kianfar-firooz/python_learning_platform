📋 متدهای لیست در پایتون

لیست یکی از پرکاربردترین دیتا تایپ‌های پایتونه که می‌تونه هر نوع داده‌ای رو در خودش ذخیره کنه. در این درس با همه متدهای لیست آشنا میشیم.

📌 لیست چیه؟

لیست یک دنباله تغییرپذیر (mutable) از اشیاءه که با [ ] تعریف میشه. می‌تونه اعضای تکراری داشته باشه و ایندکس‌گذاری از 0 شروع میشه.

# تعریف یک لیست ساده my_list = [1, "سلام", 3.14, True] print(my_list[0]) # خروجی: 1

🧰 همه متدهای لیست

.append(x)
یک عضو به انتهای لیست اضافه می‌کند
[1, 2].append(3) → [1, 2, 3]
.extend(iterable)
اعضای یک iterable را به انتها اضافه می‌کند
[1, 2].extend([3, 4]) → [1, 2, 3, 4]
.insert(i, x)
عضو x را در ایندکس i قرار می‌دهد
[1, 3].insert(1, 2) → [1, 2, 3]
.remove(x)
اولین رخداد x را حذف می‌کند
[1, 2, 1].remove(1) → [2, 1]
.pop(i=-1)
عضو ایندکس i را حذف و برمی‌گرداند
[1, 2, 3].pop(1) → 2
.clear()
همه اعضا را حذف می‌کند
[1, 2].clear() → []
.index(x, start, end)
ایندکس اولین رخداد x را برمی‌گرداند
[1, 2, 3].index(2) → 1
.count(x)
تعداد دفعات تکرار x را برمی‌گرداند
[1, 2, 2, 3].count(2) → 2
.sort(key=None, reverse=False)
لیست را مرتب می‌کند
[3, 1, 2].sort() → [1, 2, 3]
.reverse()
ترتیب اعضا را معکوس می‌کند
[1, 2, 3].reverse() → [3, 2, 1]
.copy()
یک کپی سطحی از لیست برمی‌گرداند
[1, 2].copy() → [1, 2]

🎯 مثال عملی: مدیریت لیست دانشجویان

# لیست دانشجویان students = ["علی", "مریم", "رضا"] # اضافه کردن دانشجوی جدید students.append("سارا") # حذف آخرین دانشجو last = students.pop() # مرتب کردن اسامی students.sort() print(students) # ['رضا', 'علی', 'مریم']
🏋️ تمرین:

لیستی از اعداد [5, 2, 8, 1, 9, 3] دارید. با استفاده از متدهای لیست، آن را مرتب کنید، عدد ۷ را به انتها اضافه کنید و سپس عدد ۲ را حذف کنید. کد نهایی را بنویسید.
