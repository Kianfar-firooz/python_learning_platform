📚 متدهای دیکشنری در پایتون

دیکشنری یکی از قدرتمندترین دیتا تایپ‌های پایتونه که داده‌ها رو به صورت کلید-مقدار ذخیره می‌کنه. در این درس با همه متدهای دیکشنری آشنا میشیم.

📌 دیکشنری چیه؟

دیکشنری یک دنباله تغییرپذیر (mutable) از زوج‌های کلید-مقداره که با { } تعریف میشه. کلیدها باید تغییرناپذیر (immutable) باشن و مقادیر می‌تونن هر نوع داده‌ای باشن.

# تعریف یک دیکشنری ساده person = { "name": "علی", "age": 25, "city": "تهران" } print(person["name"]) # خروجی: علی

🧰 همه متدهای دیکشنری

.clear()
همه زوج‌های کلید-مقدار را حذف می‌کند
{'a': 1}.clear() → {}
.copy()
یک کپی سطحی از دیکشنری برمی‌گرداند
{'a': 1}.copy() → {'a': 1}
.fromkeys(iterable, value)
دیکشنری جدید از کلیدها می‌سازد (متد کلاس)
dict.fromkeys(['a','b'], 0) → {'a': 0, 'b': 0}
.get(key, default)
مقدار کلید را برمی‌گرداند (در صورت نبود default)
{'a': 1}.get('b', 0) → 0
.items()
view از زوج‌های (کلید، مقدار) برمی‌گرداند
{'a': 1}.items() → dict_items([('a', 1)])
.keys()
view از کلیدها برمی‌گرداند
{'a': 1}.keys() → dict_keys(['a'])
.pop(key, default)
کلید را حذف و مقدار آن را برمی‌گرداند
{'a': 1}.pop('a') → 1
.popitem()
آخرین زوج کلید-مقدار را حذف و برمی‌گرداند
{'a':1, 'b':2}.popitem() → ('b', 2)
.setdefault(key, default)
اگر کلید نبود، با مقدار پیش‌فرض اضافه می‌کند
{'a':1}.setdefault('b',2) → {'a':1, 'b':2}
.update(other)
دیکشنری را با دیکشنری دیگر به‌روز می‌کند
{'a':1}.update({'b':2}) → {'a':1, 'b':2}
.values()
view از مقادیر برمی‌گرداند
{'a': 1}.values() → dict_values([1])

🎯 مثال عملی: مدیریت اطلاعات دانشجو

# دیکشنری اطلاعات دانشجو student = { "name": "مریم", "age": 22, "grades": [18, 19.5, 17] } # اضافه کردن کلید جدید student["university"] = "تهران" # گرفتن مقدار با get age = student.get("age", 0) # نمایش همه کلیدها print(student.keys()) # dict_keys(['name', 'age', 'grades', 'university'])
🏋️ تمرین:

یک دیکشنری از محصولات فروشگاه با کلید نام محصول و مقدار قیمت بسازید. سپس با متدهای مناسب، قیمت یک محصول را بگیرید، یک محصول جدید اضافه کنید و همه محصولات را نمایش دهید.
