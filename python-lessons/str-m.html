📝 متدهای رشته در پایتون

رشته (String) یکی از پرکاربردترین دیتا تایپ‌های پایتونه که برای کار با متن استفاده میشه. در این درس با مهم‌ترین متدهای رشته آشنا میشیم.

📌 رشته چیه؟

رشته یک دنباله تغییرناپذیر (immutable) از کاراکترهاست که با ' ' یا " " تعریف میشه. هر کاراکتر در رشته یک ایندکس داره که از 0 شروع میشه.

# تعریف یک رشته text = "سلام دنیا!" print(text[0]) # خروجی: س print(len(text)) # خروجی: 9

🧰 متدهای پرکاربرد رشته

🔤 تغییر حروف

.capitalize()
حرف اول را بزرگ، بقیه را کوچک می‌کند
"hello".capitalize() → "Hello"
.upper()
همه حروف را بزرگ می‌کند
"hello".upper() → "HELLO"
.lower()
همه حروف را کوچک می‌کند
"HELLO".lower() → "hello"
.swapcase()
کوچک↔بزرگ را عوض می‌کند
"Hello".swapcase() → "hELLO"
.title()
هر کلمه را با حرف بزرگ شروع می‌کند
"hello world".title() → "Hello World"

🔍 جستجو و پیدا کردن

.find(sub)
ایندکس اولین رخداد را برمی‌گرداند (یا -1)
"hello".find("l") → 2
.index(sub)
مثل find اما در صورت نبود خطا می‌دهد
"hello".index("e") → 1
.count(sub)
تعداد رخدادهای زیررشته را برمی‌گرداند
"hello".count("l") → 2
.startswith(prefix)
آیا با prefix شروع می‌شود؟
"hello".startswith("he") → True
.endswith(suffix)
آیا با suffix تمام می‌شود؟
"hello".endswith("lo") → True

✂️ برش و اتصال

.split(sep)
رشته را به لیست تقسیم می‌کند
"a,b,c".split(",") → ['a', 'b', 'c']
.join(iterable)
اعضا را با رشته به هم متصل می‌کند
"-".join(["a","b"]) → "a-b"
.strip()
فاصله‌های ابتدا و انتها را حذف می‌کند
" hi ".strip() → "hi"
.replace(old, new)
زیررشته را جایگزین می‌کند
"hello".replace("l","x") → "hexxo"
.partition(sep)
رشته را به ۳ بخش تقسیم می‌کند
"a-b-c".partition("-") → ('a','-','b-c')

✅ بررسی‌ها (is...)

.isalpha()
همه کاراکترها حرف هستند؟
"hello".isalpha() → True
.isdigit()
همه کاراکترها رقم هستند؟
"123".isdigit() → True
.isalnum()
همه حروف یا اعداد هستند؟
"hello123".isalnum() → True
.isspace()
همه کاراکترها فاصله هستند؟
" ".isspace() → True
.islower() / .isupper()
همه حروف کوچک/بزرگ هستند؟
"hello".islower() → True

🎯 مثال عملی: پردازش متن

# یک متن نمونه text = " سلام دنیا! " # پاکسازی متن cleaned = text.strip() # بزرگ کردن حروف upper_text = cleaned.upper() # تقسیم به کلمات words = cleaned.split() print(cleaned) # "سلام دنیا!" print(words) # ['سلام', 'دنیا!']
