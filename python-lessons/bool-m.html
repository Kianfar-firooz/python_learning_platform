✅ متدهای بولین در پایتون

بولین (Boolean) یکی از ساده‌ترین دیتا تایپ‌های پایتونه که فقط دو مقدار داره: True (درست) و False (نادرست).

📌 بولین چیه؟

بولین یک نوع داده منطقی هست که برای نشان دادن درستی یا نادرستی یک شرط استفاده میشه. در پایتون، True و False با حروف بزرگ نوشته میشن.

# مقادیر بولین is_ok = True is_error = False # نتیجه مقایسه‌ها print(5 > 3) # True print(5 == 3) # False

🧰 متدهای بولین

❌ نوع داده بولین هیچ متد خاصی ندارد!

💡 اما: می‌تونید از توابع زیر برای تبدیل به بولین استفاده کنید:
bool(0) → False
bool(42) → True
bool("") → False
bool("hi") → True
bool([]) → False
bool([1]) → True

📊 جدول تبدیل به بولین

# مقادیر False در پایتون: False, 0, 0.0, "", None, [], {}, set(), () # بقیه مقادیر همگی True هستن print(bool("سلام")) # True print(bool([1, 2])) # True print(bool({"a": 1})) # True

🎯 مثال عملی: استفاده در شرط‌ها

# بررسی وضعیت کاربر is_logged_in = True has_permission = False if is_logged_in and has_permission: print("دسترسی کامل") elif is_logged_in: print("دسترسی محدود") else: print("لطفاً وارد شوید")
🏋️ تمرین:

مقدار بولین عبارات زیر را پیش‌بینی کنید:
bool(0) → ?
bool(-5) → ?
bool("False") → ?
bool([]) → ?
bool(None) → ?

سپس کد را اجرا کنید و نتیجه را بررسی کنید.
