🎯 متدهای مجموعه در پایتون

مجموعه (Set) یک دیتا تایپ تغییرپذیر (mutable) برای ذخیره اعضای منحصر‌به‌فرد و بدون ترتیب است. در این درس با همه متدهای مجموعه آشنا میشیم.

📌 مجموعه چیه؟

مجموعه یک دنباله تغییرپذیر از اعضای منحصر‌به‌فرد که با { } تعریف میشه. اعضای مجموعه تکرار نمیشن و ترتیبی ندارن.

# تعریف یک مجموعه my_set = {1, 2, 3, 3, 2} print(my_set) # خروجی: {1, 2, 3} (تکرارها حذف شدن)

🧰 همه متدهای مجموعه

.add(x)
عضو x را به مجموعه اضافه می‌کند
{1, 2}.add(3) → {1, 2, 3}
.clear()
همه اعضا را حذف می‌کند
{1, 2}.clear() → set()
.copy()
یک کپی سطحی برمی‌گرداند
{1, 2}.copy() → {1, 2}
.difference(*sets)
تفاضل مجموعه‌ها (A - B)
{1,2,3}.difference({2,3}) → {1}
.difference_update(*sets)
تفاضل را روی خود اعمال می‌کند
s={1,2,3}; s.difference_update({2,3}) → s={1}
.discard(x)
x را حذف می‌کند (در صورت وجود)
{1, 2}.discard(3) → {1, 2}
.intersection(*sets)
اشتراک مجموعه‌ها
{1,2}.intersection({2,3}) → {2}
.intersection_update(*sets)
اشتراک را روی خود اعمال می‌کند
s={1,2}; s.intersection_update({2,3}) → s={2}
.isdisjoint(other)
آیا اشتراک ندارند؟
{1,2}.isdisjoint({3,4}) → True
.issubset(other)
زیرمجموعه است؟
{1,2}.issubset({1,2,3}) → True
.issuperset(other)
شامل مجموعه دیگر است؟
{1,2,3}.issuperset({1,2}) → True
.pop()
یک عضو دلخواه را حذف و برمی‌گرداند
{1}.pop() → 1
.remove(x)
x را حذف می‌کند (در صورت نبود خطا)
{1,2}.remove(2) → {1}
.symmetric_difference(other)
تفاضل متقارن (A △ B)
{1,2}.symmetric_difference({2,3}) → {1,3}
.symmetric_difference_update(other)
تفاضل متقارن را روی خود اعمال می‌کند
s={1,2}; s.symmetric_difference_update({2,3}) → s={1,3}
.union(*sets)
اجتماع مجموعه‌ها
{1,2}.union({2,3}) → {1,2,3}
.update(*sets)
اجتماع را روی خود اعمال می‌کند
s={1,2}; s.update({2,3}) → s={1,2,3}

🎯 مثال عملی: مدیریت مخاطبان

# مجموعه مخاطبان contacts = {"علی", "مریم", "رضا"} # اضافه کردن مخاطب جدید contacts.add("سارا") # حذف مخاطب (در صورت وجود) contacts.discard("احمد") # اشتراک با مجموعه دیگر new_contacts = {"مریم", "زهرا"} common = contacts.intersection(new_contacts) print(common) # {'مریم'}
🏋️ تمرین:

دو مجموعه زیر را در نظر بگیرید:
A = {1, 2, 3, 4, 5}
B = {4, 5, 6, 7, 8}
با استفاده از متدهای مجموعه، عملیات زیر را انجام دهید:
۱. اجتماع A و B
۲. اشتراک A و B
۳. تفاضل A از B
۴. تفاضل متقارن
