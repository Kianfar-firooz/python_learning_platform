🛡️ دستور ufw – فایروال ساده (Uncomplicated Firewall)

سطح: پیشرفته | زمان مطالعه: ۱۰ دقیقه

📖 نام (NAME)

ufw – ابزار مدیریت فایروال با رابط کاربری ساده

💻 خلاصه (SYNOPSIS)

ufw [گزینه] [دستور]

توضیح: ufw (مخفف Uncomplicated Firewall) یک ابزار ساده برای مدیریت فایروال (iptables) در اوبونتو و سایر توزیع‌های لینوکس است. این دستور برای باز و بسته کردن پورت‌ها، محدود کردن دسترسی و مدیریت امنیت شبکه بسیار مفید است.

📖 توضیح کامل (DESCRIPTION)

ufw یک رابط کاربری ساده برای فایروال iptables است که به شما اجازه می‌دهد:

  • پورت‌های خاص را باز یا بسته کنید.
  • دسترسی از یک آدرس IP خاص را محدود کنید.
  • فایروال را فعال یا غیرفعال کنید.
  • قوانین پیش‌فرض را تنظیم کنید.

نکته: قبل از استفاده از ufw، مطمئن شوید که به سرور خود از طریق SSH دسترسی دارید تا در صورت بسته شدن پورت SSH، ارتباط خود را از دست ندهید.

⚙️ دستورات اصلی

  • enable – فعال‌سازی فایروال.
  • disable – غیرفعال‌سازی فایروال.
  • status – نمایش وضعیت فایروال و قوانین فعال.
  • allow [port] – اجازه‌ی دسترسی به یک پورت.
  • deny [port] – مسدود کردن یک پورت.
  • allow from [IP] – اجازه‌ی دسترسی از یک آدرس IP خاص.
  • delete [rule] – حذف یک قانون.
  • reset – بازنشانی فایروال به حالت اولیه.
  • --help – نمایش راهنمای کوتاه.
  • --version – نمایش نسخه‌ی دستور.

🧪 مثال‌ها (EXAMPLES)

مثال ۱: نمایش وضعیت فایروال

sudo ufw status
خروجی: 
Status: inactive

وضعیت فعلی فایروال را نشان می‌دهد (غیرفعال).

مثال ۲: فعال‌سازی فایروال

sudo ufw enable
خروجی: 
Firewall is active and enabled on system startup

فایروال را فعال می‌کند.

مثال ۳: باز کردن پورت SSH (۲۲)

sudo ufw allow 22
خروجی: 
Rules updated
Rules updated (v6)

پورت ۲۲ (SSH) را باز می‌کند.

مثال ۴: باز کردن پورت HTTP (۸۰) و HTTPS (۴۴۳)

sudo ufw allow 80
sudo ufw allow 443
خروجی: 
(پورت‌های ۸۰ و ۴۴۳ باز می‌شوند)

پورت‌های وب (HTTP و HTTPS) را باز می‌کند.

مثال ۵: بستن یک پورت

sudo ufw deny 8080
خروجی: 
Rules updated
Rules updated (v6)

پورت ۸۰۸۰ را مسدود می‌کند.

مثال ۶: اجازه‌ی دسترسی فقط از یک IP خاص

sudo ufw allow from 192.168.1.50
خروجی: 
Rules updated

به آدرس IP 192.168.1.50 اجازه‌ی دسترسی کامل می‌دهد.

مثال ۷: نمایش قوانین فعال

sudo ufw status numbered
خروجی: 
Status: active

     To                         Action      From
     --                         ------      ----
[ 1] 22                         ALLOW IN    Anywhere
[ 2] 80                         ALLOW IN    Anywhere
[ 3] 443                        ALLOW IN    Anywhere
[ 4] 192.168.1.50               ALLOW IN    Anywhere

قوانین فعال را به همراه شماره نمایش می‌دهد (برای حذف مفید است).

🧠 نکته (BUGS & NOTES)

نکته: قبل از فعال‌سازی فایروال، حتماً پورت SSH را باز کنید (در غیر این صورت، دسترسی خود را به سرور از دست می‌دهید).

نکته: ufw به‌طور پیش‌فرض همه‌ی ترافیک ورودی را مسدود و همه‌ی ترافیک خروجی را مجاز می‌کند.

نکته: با sudo ufw delete [number] می‌توانید یک قانون را با شماره آن حذف کنید.

✍️ تمرین

تمرین ۱: با sudo ufw status وضعیت فایروال را ببینید.

تمرین ۲: پورت ۲۲ (SSH) را با sudo ufw allow 22 باز کنید.

تمرین ۳: فایروال را با sudo ufw enable فعال کنید.

تمرین ۴: با sudo ufw status numbered قوانین فعال را ببینید.

تمرین ۵: با sudo ufw delete یک قانون را حذف کنید.
