📦 دستور apt – نصب، حذف و مدیریت بسته‌ها

سطح: مقدماتی | زمان مطالعه: ۱۰ دقیقه

📖 نام (NAME)

apt – ابزار مدیریت بسته‌ها (نصب، حذف، جستجو و نمایش اطلاعات)

💻 خلاصه (SYNOPSIS)

sudo apt install [نام_بسته]
sudo apt remove [نام_بسته]
sudo apt purge [نام_بسته]
sudo apt search [کلمه_کلیدی]
sudo apt show [نام_بسته]

توضیح: apt برای نصب، حذف، جستجو و نمایش اطلاعات بسته‌ها استفاده می‌شود.

📖 توضیح کامل (DESCRIPTION)

apt ابزار مدیریت بسته‌های اوبونتو است که با مخازن (repositories) رسمی اوبونتو کار می‌کند. این دستور برای نصب، حذف، جستجو و نمایش اطلاعات بسته‌ها استفاده می‌شود.

عملیات‌های مهم apt:

  • install [name] – نصب یک بسته از مخازن.
  • remove [name] – حذف یک بسته (فایل‌های تنظیمات باقی می‌مانند).
  • purge [name] – حذف کامل بسته به همراه فایل‌های تنظیمات.
  • search [keyword] – جستجوی بسته‌ها با کلمه‌ی کلیدی.
  • show [name] – نمایش اطلاعات کامل یک بسته (نسخه، توضیح، وابستگی‌ها).
  • autoremove – حذف بسته‌های اضافی که دیگر نیازی نیستند.
  • list --installed – نمایش لیست بسته‌های نصب‌شده.

🧪 مثال‌ها (EXAMPLES)

مثال ۱: نصب یک بسته

sudo apt install htop
خروجی: 
Reading package lists... Done
Building dependency tree... Done
The following NEW packages will be installed:
  htop
0 upgraded, 1 newly installed, 0 to remove and 0 not upgraded.
Need to get 76.8 kB of archives.
After this operation, 232 kB of additional disk space will be used.
Do you want to continue? [Y/n]

برنامه‌ی htop را از مخازن نصب می‌کند. با Y تأیید کنید.

مثال ۲: نصب چند بسته با یک دستور

sudo apt install nano vim git
خروجی: 
(سه بسته به طور همزمان نصب می‌شوند)

سه بسته‌ی nano، vim و git را با یک دستور نصب می‌کند.

مثال ۳: حذف یک بسته (remove)

sudo apt remove htop
خروجی: 
Reading package lists... Done
Building dependency tree... Done
The following packages will be REMOVED:
  htop
0 upgraded, 0 newly installed, 1 to remove and 0 not upgraded.
After this operation, 232 kB disk space will be freed.
Do you want to continue? [Y/n]

htop را حذف می‌کند، اما فایل‌های تنظیمات آن باقی می‌مانند.

مثال ۴: حذف کامل یک بسته (purge)

sudo apt purge htop
خروجی: 
Reading package lists... Done
Building dependency tree... Done
The following packages will be REMOVED:
  htop*
0 upgraded, 0 newly installed, 1 to remove and 0 not upgraded.
After this operation, 232 kB disk space will be freed.
Do you want to continue? [Y/n]

htop را به همراه فایل‌های تنظیمات آن حذف می‌کند (علامت * نشان‌دهنده‌ی حذف کامل است).

مثال ۵: جستجوی یک بسته

apt search "text editor"
خروجی: 
vim - Vi IMproved - enhanced vi editor
nano - small, friendly text editor inspired by Pico
gedit - official text editor of the GNOME desktop environment
...

بسته‌های مربوط به "text editor" را جستجو می‌کند.

مثال ۶: نمایش اطلاعات یک بسته

apt show htop
خروجی: 
Package: htop
Version: 2.2.0-2build1
Priority: optional
Section: utils
Maintainer: Ubuntu Developers
Installed-Size: 232 kB
Depends: libc6, libncursesw6, libtinfo6
Description: interactive processes viewer
 This is htop, an interactive process viewer for Linux.
...

اطلاعات کامل بسته‌ی htop را نمایش می‌دهد.

مثال ۷: حذف بسته‌های اضافی (autoremove)

sudo apt autoremove
خروجی: 
Reading package lists... Done
Building dependency tree... Done
The following packages will be REMOVED:
  libpython3.8-minimal
0 upgraded, 0 newly installed, 1 to remove and 0 not upgraded.
Do you want to continue? [Y/n]

بسته‌هایی که به عنوان وابستگی نصب شده‌اند ولی دیگر به آن‌ها نیازی نیست را حذف می‌کند.

🧠 نکته (BUGS & NOTES)

نکته: برای نصب یا حذف بسته‌ها حتماً از sudo استفاده کنید.

نکته: قبل از نصب یک بسته، با apt search یا apt show اطلاعات آن را بررسی کنید.

نکته: بین remove و purge تفاوت وجود دارد: purge فایل‌های تنظیمات را هم حذف می‌کند.

✍️ تمرین

تمرین ۱: با apt search یک برنامه‌ی دلخواه را جستجو کنید.

تمرین ۲: با sudo apt install برنامه‌ی nano را نصب کنید.

تمرین ۳: با apt show اطلاعات بسته‌ی nano را ببینید.

تمرین ۴: با sudo apt remove برنامه‌ی نصب‌شده را حذف کنید.

تمرین ۵: با sudo apt autoremove بسته‌های اضافی را حذف کنید.
