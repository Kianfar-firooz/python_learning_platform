🗑️ دستور rm – حذف فایل‌ها و پوشه‌ها

سطح: مقدماتی | زمان مطالعه: ۸ دقیقه

📖 نام (NAME)

rm – حذف فایل‌ها یا پوشه‌ها

💻 خلاصه (SYNOPSIS)

rm [گزینه] [نام فایل/پوشه]

توضیح: یک یا چند فایل یا پوشه را به طور دائمی حذف می‌کند. این دستور بسیار قدرتمند است و باید با احتیاط استفاده شود.

📖 توضیح کامل (DESCRIPTION)

دستور rm (مخفف Remove) برای حذف فایل‌ها و پوشه‌ها از سیستم فایل استفاده می‌شود. برخلاف سطل زباله در ویندوز، فایل‌های حذف شده با rm به طور کامل از سیستم حذف می‌شوند و قابل بازیابی نیستند (مگر با ابزارهای خاص).

به همین دلیل، استفاده از این دستور نیاز به دقت بالایی دارد.

⚙️ لیست کامل گزینه‌ها (OPTIONS)

  • -f – حذف اجباری (ignore nonexistent files, never prompt).
  • -i – قبل از هر حذف، تأیید می‌گیرد (برای هر فایل جداگانه).
  • -I – قبل از حذف بیش از ۳ فایل یا حذف بازگشتی، یک بار تأیید می‌گیرد.
  • -r – حذف بازگشتی (برای حذف پوشه‌ها و محتویاتشان استفاده می‌شود).
  • -R – مشابه -r (همان حذف بازگشتی).
  • -v – نمایش جزئیات حذف (verbose).
  • -d – حذف پوشه‌های خالی (با -r تفاوت دارد).
  • -r – با -f ترکیب می‌شود و بدون هیچ تأییدی، همه‌چیز را حذف می‌کند.
  • --help – نمایش راهنمای کوتاه.
  • --version – نمایش نسخه‌ی دستور.

🧪 مثال‌ها (EXAMPLES)

مثال ۱: حذف یک فایل

ساده‌ترین حالت استفاده از rm:

rm file.txt
خروجی: 
(هیچ خروجی، فایل حذف می‌شود)

فایل file.txt را حذف می‌کند.

مثال ۲: حذف چند فایل با یک دستور

می‌توانید چندین فایل را با یک rm حذف کنید:

rm file1.txt file2.txt file3.txt
خروجی: 
(سه فایل حذف می‌شوند)

هر سه فایل را به طور همزمان حذف می‌کند.

مثال ۳: حذف با تأیید (ایمن‌تر)

با گزینه‌ی -i، قبل از هر حذف تأیید می‌گیرد:

rm -i file.txt
خروجی: 
rm: remove regular file 'file.txt'? (y/n)

با y تأیید و با n لغو می‌کنید.

مثال ۴: حذف یک پوشه و محتویاتش (بازگشتی)

با گزینه‌ی -r، کل پوشه و همه‌ی محتویات آن حذف می‌شود:

rm -r myfolder
خروجی: 
(همه‌ی محتویات پوشه حذف می‌شود)

پوشه‌ی myfolder و تمام فایل‌ها و زیرپوشه‌های آن را حذف می‌کند.

مثال ۵: حذف پوشه با تأیید (ایمن‌تر)

ترکیب -ri برای حذف بازگشتی با تأیید:

rm -ri myfolder
خروجی: 
rm: descend into directory 'myfolder'? (y/n)
rm: remove 'myfolder/file1.txt'? (y/n)
rm: remove 'myfolder/file2.txt'? (y/n)
rm: remove directory 'myfolder'? (y/n)

قبل از ورود به پوشه و قبل از حذف هر فایل، تأیید می‌گیرد.

مثال ۶: حذف اجباری (خطرناک!)

با ترکیب -rf، همه‌چیز بدون هیچ تأییدی حذف می‌شود:

rm -rf myfolder
خروجی: 
(همه‌چیز حذف می‌شود، بدون سوال)

⚠️ هشدار: از این ترکیب بسیار بسیار با احتیاط استفاده کنید. یک اشتباه می‌تواند کل سیستم شما را حذف کند (مثلاً rm -rf /).

📘 همچنین ببینید (SEE ALSO)

  • man rmdir – حذف پوشه‌های خالی
  • man unlink – حذف یک فایل با استفاده از سیستم‌کال unlink
  • man shred – حذف ایمن فایل‌ها (با بازنویسی محتوا)

✍️ تمرین

تمرین ۱: یک فایل به نام test.txt بسازید و سپس آن را با rm حذف کنید.

تمرین ۲: یک پوشه به نام testdir بسازید و داخل آن چند فایل خالی ایجاد کنید. سپس کل پوشه را با rm -r حذف کنید.

تمرین ۳: با گزینه‌ی -i یک فایل را حذف کنید و ببینید چه پیامی می‌دهد.
