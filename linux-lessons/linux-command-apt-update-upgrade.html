📦 دستور apt – به‌روزرسانی لیست بسته‌ها و سیستم

سطح: مقدماتی | زمان مطالعه: ۸ دقیقه

📖 نام (NAME)

apt – ابزار مدیریت بسته‌های خط فرمان در اوبونتو و دبیان

💻 خلاصه (SYNOPSIS)

sudo apt update
sudo apt upgrade

توضیح: apt ابزار مدیریت بسته‌های اوبونتو است. با update لیست بسته‌های موجود در مخازن را به‌روز می‌کنید و با upgrade بسته‌های نصب‌شده را به آخرین نسخه به‌روز می‌کنید.

📖 توضیح کامل (DESCRIPTION)

apt (مخفف Advanced Package Tool) ابزار اصلی مدیریت بسته‌ها در اوبونتو و دبیان است. این دستور با مخازن (repositories) رسمی اوبونتو کار می‌کند و به شما امکان نصب، حذف، به‌روزرسانی و جستجوی برنامه‌ها را می‌دهد.

دو عملیات اساسی update و upgrade برای نگهداری سیستم ضروری هستند:

  • update: لیست بسته‌های موجود در مخازن را به‌روز می‌کند (بدون نصب چیزی).
  • upgrade: تمام بسته‌های نصب‌شده را به آخرین نسخه‌ی موجود به‌روز می‌کند.

⚙️ لیست کامل گزینه‌ها (OPTIONS) برای apt

  • -y – پاسخ خودکار "بله" به سوالات (برای اجرای بدون دخالت کاربر).
  • -s – شبیه‌سازی (simulate) بدون انجام تغییرات واقعی.
  • -q – حالت بی‌صدا (quiet) – نمایش خروجی کمتر.
  • -v – نمایش نسخه‌ی apt.
  • -h – نمایش راهنمای کوتاه.
  • --no-upgrade – از به‌روزرسانی بسته‌ها جلوگیری می‌کند.
  • --only-upgrade – فقط بسته‌هایی که قبلاً نصب شده‌اند را به‌روز می‌کند.
  • --allow-downgrades – اجازه‌ی پایین‌آوردن نسخه‌ی بسته‌ها را می‌دهد.

🧪 مثال‌ها (EXAMPLES)

مثال ۱: به‌روزرسانی لیست بسته‌ها (update)

اولین قدم برای هر عملیات apt، اجرای update است:

sudo apt update
خروجی: 
Hit:1 http://archive.ubuntu.com/ubuntu focal InRelease
Get:2 http://archive.ubuntu.com/ubuntu focal-updates InRelease [89.1 kB]
Get:3 http://archive.ubuntu.com/ubuntu focal-backports InRelease [89.1 kB]
...
Reading package lists... Done

لیست بسته‌های موجود در مخازن را به‌روز می‌کند. این کار را همیشه قبل از نصب یا upgrade انجام دهید.

مثال ۲: به‌روزرسانی تمام بسته‌ها (upgrade)

بعد از update، با upgrade بسته‌ها را به‌روز کنید:

sudo apt upgrade
خروجی: 
Reading package lists... Done
Building dependency tree... Done
Calculating upgrade... Done
The following packages will be upgraded:
  bash nano vim
3 upgraded, 0 newly installed, 0 to remove and 0 not upgraded.
Need to get 3,245 kB of archives.
After this operation, 12.5 kB of additional disk space will be used.
Do you want to continue? [Y/n]

تمام بسته‌هایی که نسخه‌ی جدید دارند را به‌روز می‌کند. با Y تأیید کنید.

مثال ۳: به‌روزرسانی خودکار با پاسخ بله

با گزینه‌ی -y، بدون نیاز به تأیید، همه‌چیز به‌روز می‌شود:

sudo apt upgrade -y
خروجی: 
(همان خروجی upgrade، اما بدون سوال و به‌طور خودکار ادامه می‌دهد)

مناسب برای اسکریپت‌ها و مواقعی که نمی‌خواهید دستی تأیید کنید.

مثال ۴: به‌روزرسانی کامل (با حذف بسته‌های اضافی)

با full-upgrade (یا dist-upgrade) می‌توانید بسته‌های اضافی را هم حذف کنید:

sudo apt full-upgrade
خروجی: 
Reading package lists... Done
Building dependency tree... Done
Calculating upgrade... Done
The following packages will be REMOVED:
  old-package
The following packages will be upgraded:
  ...
Do you want to continue? [Y/n]

این گزینه ممکن است بسته‌های اضافی را حذف کند تا وابستگی‌ها برطرف شوند.

🧠 نکته (BUGS & NOTES)

نکته: همیشه اول sudo apt update و بعد sudo apt upgrade را اجرا کنید.

نکته: برای به‌روزرسانی هسته و تغییرات بزرگ، از full-upgrade استفاده کنید.

نکته: بعد از upgrade، اگر هسته به‌روز شده باشد، برای اعمال تغییرات باید سیستم را ری‌استارت کنید.

✍️ تمرین

تمرین ۱: با sudo apt update لیست بسته‌ها را به‌روز کنید.

تمرین ۲: با sudo apt upgrade بسته‌های سیستم را به‌روز کنید.

تمرین ۳: با sudo apt upgrade -y به‌روزرسانی خودکار را امتحان کنید.
