💻 ترمینال چیه و چطور بازش کنیم؟

سطح: مقدماتی | زمان مطالعه: ۸ دقیقه

📖 ترمینال (خط فرمان) چیه؟

ترمینال یه پنجره‌ی سیاه هست که توش با نوشتن دستور، به کامپیوتر می‌گیم چه کاری انجام بده. درست مثل وقتی که به یه نفر دستور می‌دی، اما اینجا به کامپیوتر دستور می‌دی.

توی لینوکس، خیلی از کارها با ترمینال سریع‌تر و راحت‌تر انجام میشه.

🖱️ چطور ترمینال رو باز کنیم؟

توی اوبونتو، کافیه کلیدهای Ctrl + Alt + T رو همزمان بزنی. یا اینکه از منوی برنامه‌ها، کلمه‌ی Terminal رو جستجو کنی و روش کلیک کنی.

💡 نکته: ترمینال رو با اسم‌های Terminal، Console یا Command Line هم میشناسن.

🧪 اولین دستورت رو تایپ کن

بعد از باز شدن ترمینال، این دستور رو تایپ کن و بعد Enter بزن:

whoami
خروجی: 
username

این دستور اسم کاربری تو رو نشون میده. یعنی همون اسمی که موقع نصب اوبونتو برای خودت انتخاب کردی.

✍️ تمرین

تمرین: ترمینال رو باز کن و دستور whoami رو بزن. ببین اسم کاربریت چیه؟ (اگه جوابش رو توی خروجی دیدی، یعنی ترمینال درست کار می‌کنه!)

🧠 نکته

دستورات لینوکس معمولاً به حروف کوچک و بزرگ حساس هستن. یعنی whoami با WhoAmI فرق داره و ممکنه دومی خطا بده.
