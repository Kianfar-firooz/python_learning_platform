📂 دستور ls – لیست گرفتن از فایل‌ها و پوشه‌ها

سطح: مقدماتی | زمان مطالعه: ۱۵ دقیقه

📖 نام (NAME)

ls – لیست گرفتن از محتویات یک پوشه

💻 خلاصه (SYNOPSIS)

ls [گزینه] [فایل/پوشه]

توضیح: گزینه کارهایی که دستور باید انجام بده رو مشخص میکنه. فایل/پوشه مسیری هست که میخوای لیستش رو ببینی. اگه ندی، پوشه‌ی فعلی رو نشون میده.

📖 توضیح کامل (DESCRIPTION)

دستور ls اطلاعات مربوط به فایل‌ها و پوشه‌ها رو نشون میده. به‌طور پیش‌فرض، اسم فایل‌ها رو به ترتیب حروف الفبا لیست میکنه. اگه به جای فایل، یک پوشه بهش بدی، محتویات اون پوشه رو لیست میکنه. اگه چیزی بهش ندیا، پوشه‌ی فعلی رو لیست میکنه.

⚙️ لیست کامل گزینه‌ها (OPTIONS)

  • -l – نمایش جزئیات کامل (حجم، تاریخ، مالک، مجوزها، تعداد لینک‌ها)
  • -a – نمایش همه‌ی فایل‌ها، حتی فایل‌های مخفی (با نقطه شروع میشن)
  • -h – نمایش حجم به صورت قابل‌خوانی (مثلاً 1K، 1M، 1G) – معمولاً با -l استفاده میشه
  • -R – نمایش همه‌ی محتویات پوشه‌ها به صورت بازگشتی (زیرپوشه‌ها هم)
  • -S – مرتب‌سازی بر اساس حجم (بزرگ‌ترین اول)
  • -t – مرتب‌سازی بر اساس تاریخ تغییر (جدیدترین اول)
  • -r – برعکس کردن ترتیب مرتب‌سازی (مثلاً با -t قدیمی‌ترین اول)
  • -i – نمایش شماره‌ی inode (شناسه‌ی یکتا برای هر فایل)
  • -n – نمایش ID عددی کاربر و گروه به جای اسم
  • -F – اضافه کردن علامت به انتهای اسم‌ها (مثل / برای پوشه و * برای فایل‌های اجرایی)
  • -p – اضافه کردن / به انتهای اسم پوشه‌ها
  • -d – نمایش خود پوشه، نه محتویاتش (معمولاً با -l استفاده میشه)
  • -1 – نمایش هر فایل در یک خط جداگانه (حالت ستونی)
  • -m – نمایش فایل‌ها به صورت جدا شده با کاما
  • -Q – قرار دادن اسم فایل‌ها بین دو علامت کوتیشن ("")
  • --help – نمایش راهنمای کوتاه
  • --version – نمایش نسخه‌ی دستور

گزینه‌های مرتب‌سازی:

  • -f – غیرفعال کردن مرتب‌سازی (همون ترتیبی که توی پوشه هست)
  • -U – مرتب‌سازی بر اساس ترتیب ایجاد فایل (زمان خلق)
  • -c – مرتب‌سازی بر اساس زمان تغییر وضعیت (ctime) با -lt
  • -u – مرتب‌سازی بر اساس زمان آخرین دسترسی (atime) با -lt

گزینه‌های نمایش:

  • -C – نمایش به صورت چند ستونی (حالت پیش‌فرض)
  • -x – نمایش به صورت چند ستونی ولی به ترتیب سطری
  • -v – مرتب‌سازی بر اساس نسخه (مثلاً file1, file2, file10)
  • -X – مرتب‌سازی بر اساس پسوند فایل

گزینه‌های مربوط به رنگ:

  • --color – نمایش با رنگ (مثلاً --color=auto)
  • --color=never – غیرفعال کردن رنگ
  • --color=always – همیشه نمایش با رنگ

🧪 مثال‌ها (EXAMPLES)

مثال ۱: ساده‌ترین حالت

ls
خروجی: 
Documents  Downloads  Music  Pictures  Videos

فقط اسم فایل‌ها و پوشه‌های پوشه‌ی فعلی رو نشون میده.

مثال ۲: نمایش جزئیات کامل

ls -l
خروجی: 
drwxr-xr-x 2 user user 4096 Jul 20 10:00 Documents
-rw-r--r-- 1 user user  1024 Jul 19 15:30 note.txt

توضیح ستون‌ها:

  • ستون اول: مجوزها (نوع فایل و دسترسی‌ها)
  • ستون دوم: تعداد لینک‌های سخت
  • ستون سوم: نام مالک
  • ستون چهارم: نام گروه
  • ستون پنجم: حجم (به بایت)
  • ستون ششم تا هشتم: تاریخ آخرین تغییر
  • ستون نهم: اسم فایل یا پوشه

مثال ۳: نمایش فایل‌های مخفی

ls -a
خروجی: 
.  ..  .bashrc  .profile  Documents  Downloads

فایل‌هایی که با نقطه شروع میشن (مثل .bashrc) رو هم نشون میده. . یعنی پوشه‌ی فعلی و .. یعنی پوشه‌ی بالاتر.

مثال ۴: نمایش حجم قابل‌خوانی

ls -lh
خروجی: 
drwxr-xr-x 2 user user 4.0K Jul 20 10:00 Documents
-rw-r--r-- 1 user user 1.0K Jul 19 15:30 note.txt

حجم فایل‌ها رو به صورت قابل‌خوانی (K, M, G) نشون میده.

مثال ۵: نمایش همه‌چیز با جزئیات و مخفی‌ها

ls -la
خروجی: 
total 48
drwxr-xr-x 5 user user 4096 Jul 20 10:00 .
drwxr-xr-x 3 root root 4096 Jul 19 08:00 ..
-rw-r--r-- 1 user user  220 Jul 19 08:00 .bash_logout
-rw-r--r-- 1 user user 3771 Jul 19 08:00 .bashrc
drwxr-xr-x 2 user user 4096 Jul 20 10:00 Documents
-rw-r--r-- 1 user user  1024 Jul 19 15:30 note.txt

ترکیب -l و -a برای نمایش همه‌ی فایل‌ها با جزئیات کامل.

🧠 نکته (BUGS & NOTES)

نکته: بعضی از گزینه‌ها ممکنه توی نسخه‌های مختلف لینوکس کمی فرق کنن. برای دیدن گزینه‌های دقیق نسخه‌ی خودت، از ls --help استفاده کن.

نکته: گزینه‌های -l و -a رو می‌تونی به هم بچسبونی: ls -la یا ls -al هر دو یک کار رو انجام میدن.

📚 همچنین ببینید (SEE ALSO)

  • man pwd – دیدن مسیر فعلی
  • man cd – تغییر پوشه
  • man tree – نمایش درختی پوشه‌ها
  • man stat – اطلاعات دقیق فایل

✍️ تمرین

تمرین ۱: ترمینال رو باز کن و توی پوشه‌ی /etc برو. با ls -l لیست فایل‌ها رو ببین.

تمرین ۲: با ls -la فایل‌های مخفی رو هم توی /etc ببین.

تمرین ۳: با ls -lhS فایل‌ها رو بر اساس حجم (بزرگ‌ترین اول) و با حجم قابل‌خوانی ببین.

تمرین ۴: با ls -R محتویات پوشه‌ی فعلی رو به صورت بازگشتی ببین.
