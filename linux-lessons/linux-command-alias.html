⚡ دستور alias – ایجاد میانبر برای دستورات

سطح: متوسط | زمان مطالعه: ۸ دقیقه

📖 نام (NAME)

alias – ایجاد نام مستعار (میانبر) برای دستورات یا ترکیب دستورات

💻 خلاصه (SYNOPSIS)

alias [نام=دستور]

توضیح: alias به شما اجازه می‌دهد برای دستورات طولانی یا پیچیده، یک نام کوتاه‌تر ایجاد کنید. این کار باعث افزایش سرعت کار و کاهش اشتباهات تایپی می‌شود.

📖 توضیح کامل (DESCRIPTION)

دستور alias یک میانبر (نام مستعار) برای یک دستور یا ترکیب دستورات ایجاد می‌کند. این میانبر فقط در همان جلسه‌ی ترمینال (Session) معتبر است و با بستن ترمینال از بین می‌رود. برای دائمی کردن آن، باید به فایل .bashrc اضافه شود.

aliasها به شما کمک می‌کنند تا دستورات طولانی را سریع‌تر تایپ کنید و از اشتباهات جلوگیری کنید.

⚙️ گزینه‌ها (OPTIONS)

  • -p – نمایش لیست همه‌ی aliasهای تعریف‌شده.
  • --help – نمایش راهنمای کوتاه.
  • --version – نمایش نسخه‌ی دستور.

🧪 مثال‌ها (EXAMPLES)

مثال ۱: ایجاد یک alias ساده

alias ll='ls -l'
خروجی: 
(هیچ خروجی، alias تعریف می‌شود)

با تایپ ll، دستور ls -l اجرا می‌شود.

مثال ۲: ایجاد alias با چند دستور

alias update='sudo apt update && sudo apt upgrade -y'
خروجی: 
(هیچ خروجی، alias تعریف می‌شود)

با تایپ update، سیستم به‌روز می‌شود.

مثال ۳: نمایش لیست aliasهای تعریف‌شده

alias -p
خروجی: 
alias ll='ls -l'
alias update='sudo apt update && sudo apt upgrade -y'
alias la='ls -la'

همه‌ی aliasهای تعریف‌شده را نمایش می‌دهد.

مثال ۴: حذف یک alias

unalias ll
خروجی: 
(هیچ خروجی، alias ll حذف می‌شود)

alias ll را حذف می‌کند.

مثال ۵: دائمی کردن alias در .bashrc

برای دائمی کردن alias، آن را به فایل .bashrc اضافه کنید:

echo "alias ll='ls -l'" >> ~/.bashrc
source ~/.bashrc
خروجی: 
(هیچ خروجی، alias دائمی می‌شود)

با این کار، هر بار که ترمینال را باز می‌کنید، alias در دسترس خواهد بود.

🧠 نکته (BUGS & NOTES)

نکته: aliasها در یک جلسه‌ی ترمینال معتبر هستند و با بستن ترمینال از بین می‌روند.

نکته: برای دائمی کردن alias، آن را به فایل ~/.bashrc یا ~/.bash_aliases اضافه کنید.

نکته: اگر alias شما با یک دستور دیگر هم‌نام باشد، دستور اصلی اجرا نمی‌شود (برای اجرای دستور اصلی از \command استفاده کنید).

✍️ تمرین

تمرین ۱: یک alias برای دستور ls -la تعریف کنید.

تمرین ۲: یک alias برای ترکیب sudo apt update && sudo apt upgrade -y تعریف کنید.

تمرین ۳: با alias -p لیست aliasها را ببینید.

تمرین ۴: یک alias را با unalias حذف کنید.
