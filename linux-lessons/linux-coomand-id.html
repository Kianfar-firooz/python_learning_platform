🆔 دستور id – نمایش اطلاعات کاربر و گروه

سطح: مقدماتی | زمان مطالعه: ۶ دقیقه

📖 نام (NAME)

id – چاپ اطلاعات کاربر و گروه‌های مرتبط

💻 خلاصه (SYNOPSIS)

id [گزینه] [نام کاربر]

توضیح: اطلاعات مربوط به کاربر (مانند UID، GID و گروه‌های عضو) را نمایش می‌دهد. اگر نام کاربری داده نشود، اطلاعات کاربر فعلی نمایش داده می‌شود.

📖 توضیح کامل (DESCRIPTION)

دستور id اطلاعات کامل یک کاربر را نمایش می‌دهد:

  • UID (User ID): شناسه‌ی عددی کاربر.
  • GID (Group ID): شناسه‌ی گروه اصلی کاربر.
  • گروه‌ها: لیست گروه‌هایی که کاربر در آن‌ها عضو است.

این دستور برای مدیریت کاربران و گروه‌ها بسیار مفید است.

⚙️ لیست کامل گزینه‌ها (OPTIONS)

  • -u – فقط UID (شناسه کاربر) را نمایش می‌دهد.
  • -g – فقط GID (شناسه گروه اصلی) را نمایش می‌دهد.
  • -G – شناسه‌های تمام گروه‌ها را نمایش می‌دهد.
  • -n – به جای شماره، نام را نمایش می‌دهد (با -u، -g یا -G استفاده شود).
  • -r – اطلاعات واقعی را نمایش می‌دهد (نه مؤثر).
  • -z – خروجی را با کاراکتر null جدا می‌کند.
  • --help – نمایش راهنمای کوتاه.
  • --version – نمایش نسخه‌ی دستور.

🧪 مثال‌ها (EXAMPLES)

مثال ۱: نمایش اطلاعات کامل کاربر فعلی

id
خروجی: 
uid=1000(alice) gid=1000(alice) groups=1000(alice),27(sudo),1001(developers)

شناسه (UID) و گروه (GID) کاربر alice و گروه‌های دیگری که در آن‌ها عضو است را نمایش می‌دهد.

مثال ۲: نمایش اطلاعات یک کاربر دیگر

id bob
خروجی: 
uid=1001(bob) gid=1001(bob) groups=1001(bob),1002(designers)

اطلاعات کاربر bob را نمایش می‌دهد.

مثال ۳: نمایش فقط UID

id -u
خروجی: 
1000

فقط شناسه‌ی عددی کاربر فعلی را نمایش می‌دهد.

مثال ۴: نمایش نام کاربر با UID

id -nu
خروجی: 
alice

نام کاربر فعلی را نمایش می‌دهد (معادل whoami).

مثال ۵: نمایش همه‌ی گروه‌ها

id -G
خروجی: 
1000 27 1001

شناسه‌های عددی تمام گروه‌هایی که کاربر در آن‌ها عضو است را نمایش می‌دهد.

🧠 نکته (BUGS & NOTES)

نکته: id -nu معادل whoami است.

نکته: کاربر root همیشه UID=0 دارد.

نکته: با id می‌توانید بفهمید که آیا یک کاربر در یک گروه خاص عضو است یا نه.

✍️ تمرین

تمرین ۱: با id اطلاعات کامل خود را ببینید.

تمرین ۲: با id -u UID خود را ببینید.

تمرین ۳: با id -nu نام کاربری خود را ببینید.

تمرین ۴: اطلاعات کاربر root را با id root ببینید.
