💾 دستور df – نمایش فضای دیسک

سطح: متوسط | زمان مطالعه: ۷ دقیقه

📖 نام (NAME)

df – گزارش از فضای استفاده‌شده و آزاد دیسک

💻 خلاصه (SYNOPSIS)

df [گزینه] [مسیر]

توضیح: فضای استفاده‌شده و آزاد دیسک را در سیستم فایل‌های مختلف نمایش می‌دهد.

📖 توضیح کامل (DESCRIPTION)

دستور df (مخفف Disk Free) اطلاعاتی درباره‌ی فضای دیسک در تمام سیستم‌فایل‌های نصب‌شده (مانند /، /home، /var و ...) ارائه می‌دهد.

⚙️ لیست کامل گزینه‌ها (OPTIONS)

  • -h – نمایش اعداد به صورت قابل خواندن (human-readable).
  • -H – نمایش اعداد با پایه ۱۰۰۰ (به جای ۱۰۲۴).
  • -T – نمایش نوع سیستم فایل (مثل ext4, xfs).
  • -t – نمایش فقط سیستم‌فایل‌های یک نوع خاص (مثلاً -t ext4).
  • -x – نمایش همه‌ی سیستم‌فایل‌ها به جز یک نوع خاص.
  • --total – نمایش مجموع فضای دیسک.
  • -i – نمایش استفاده از inodes به جای فضای دیسک.
  • --help – نمایش راهنمای کوتاه.
  • --version – نمایش نسخه‌ی دستور.

🧪 مثال‌ها (EXAMPLES)

مثال ۱: نمایش فضای دیسک

df -h
خروجی: 
Filesystem      Size  Used Avail Use% Mounted on
/dev/sda1       100G   50G   50G  50% /
/dev/sdb1        50G   20G   30G  40% /home

فضای دیسک را در تمام سیستم‌فایل‌ها به صورت قابل خواندن نمایش می‌دهد.

مثال ۲: نمایش فضای دیسک با نوع سیستم فایل

df -hT
خروجی: 
Filesystem     Type      Size  Used Avail Use% Mounted on
/dev/sda1      ext4      100G   50G   50G  50% /
/dev/sdb1      ext4       50G   20G   30G  40% /home

نوع سیستم فایل (مثل ext4) را نیز نمایش می‌دهد.

مثال ۳: نمایش فضای یک مسیر خاص

df -h /home
خروجی: 
Filesystem      Size  Used Avail Use% Mounted on
/dev/sdb1        50G   20G   30G  40% /home

فقط فضای دیسک مربوط به پوشه‌ی /home را نمایش می‌دهد.

مثال ۴: نمایش مجموع فضای دیسک

df -h --total
خروجی: 
Filesystem      Size  Used Avail Use% Mounted on
/dev/sda1       100G   50G   50G  50% /
/dev/sdb1        50G   20G   30G  40% /home
total           150G   70G   80G  47% -

مجموع فضای دیسک را در انتهای خروجی نمایش می‌دهد.

🧠 نکته (BUGS & NOTES)

نکته: اگر فضای دیسک کم است، از دستور du برای پیدا کردن پوشه‌های حجیم استفاده کنید.

نکته: df فقط فضای سیستم‌فایل‌های نصب‌شده را نشان می‌دهد.

✍️ تمرین

تمرین ۱: با df -h فضای دیسک را ببینید.

تمرین ۲: با df -hT نوع سیستم‌فایل را نیز مشاهده کنید.

تمرین ۳: با df -h --total مجموع فضای دیسک را ببینید.
