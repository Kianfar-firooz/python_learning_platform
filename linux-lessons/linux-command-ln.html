🔗 دستور ln – ایجاد لینک‌های سمبلیک و سخت

سطح: متوسط | زمان مطالعه: ۱۰ دقیقه

📖 نام (NAME)

ln – ایجاد لینک (ارتباط) بین فایل‌ها و پوشه‌ها

💻 خلاصه (SYNOPSIS)

ln [گزینه] [هدف] [نام لینک]

توضیح: ln برای ایجاد لینک‌های سمبلیک (Symbolic Links) و لینک‌های سخت (Hard Links) استفاده می‌شود. لینک‌های سمبلیک مانند میانبرهای ویندوز عمل می‌کنند و به یک فایل یا پوشه اشاره می‌کنند.

📖 توضیح کامل (DESCRIPTION)

دستور ln (مخفف Link) برای ایجاد ارتباط بین فایل‌ها و پوشه‌ها استفاده می‌شود. دو نوع لینک در لینوکس وجود دارد:

  • لینک سخت (Hard Link): یک کپی اضافی از فایل است که به همان inode (شناسه‌ی فایل) اشاره می‌کند. اگر فایل اصلی حذف شود، لینک سخت همچنان معتبر است.
  • لینک سمبلیک (Symbolic Link): مانند میانبر است و به مسیر فایل اشاره می‌کند. اگر فایل اصلی حذف شود، لینک سمبلیک می‌شکند.

⚙️ لیست کامل گزینه‌ها (OPTIONS)

  • -s – ایجاد لینک سمبلیک (نمادین).
  • -f – بازنویسی لینک‌های موجود (force).
  • -n – اگر نام لینک یک پوشه باشد، آن را به‌روز می‌کند.
  • -i – قبل از بازنویسی، تأیید می‌گیرد.
  • -v – نمایش جزئیات (verbose).
  • -b – قبل از بازنویسی، از فایل مقصد بک‌آپ می‌گیرد.
  • -L – لینک سخت را از یک لینک سمبلیک دنبال می‌کند.
  • -P – لینک سخت را از خود لینک سمبلیک می‌سازد.
  • --help – نمایش راهنمای کوتاه.
  • --version – نمایش نسخه‌ی دستور.

🧪 مثال‌ها (EXAMPLES)

مثال ۱: ایجاد لینک سمبلیک به یک فایل

ln -s /usr/bin/python3 python
خروجی: 
(هیچ خروجی، لینک سمبلیک python ساخته می‌شود)

یک لینک سمبلیک به نام python می‌سازد که به /usr/bin/python3 اشاره می‌کند.

مثال ۲: ایجاد لینک سمبلیک به یک پوشه

ln -s /home/user/Documents/ mydocs
خروجی: 
(هیچ خروجی، لینک سمبلیک mydocs ساخته می‌شود)

یک لینک سمبلیک به نام mydocs می‌سازد که به پوشه‌ی Documents اشاره می‌کند.

مثال ۳: ایجاد لینک سخت

ln file.txt hardlink.txt
خروجی: 
(هیچ خروجی، لینک سخت hardlink.txt ساخته می‌شود)

یک لینک سخت به نام hardlink.txt می‌سازد که همان محتوای file.txt را دارد.

مثال ۴: نمایش لینک‌های سمبلیک با ls -l

ls -l python
خروجی: 
lrwxrwxrwx 1 user user 16 Jul 20 10:00 python -> /usr/bin/python3

با ls -l، لینک‌های سمبلیک با -> مشخص می‌شوند.

مثال ۵: شکستن لینک سمبلیک

rm python
خروجی: 
(هیچ خروجی، لینک سمبلیک حذف می‌شود)

لینک سمبلیک را حذف می‌کند (فایل اصلی حذف نمی‌شود).

🧠 نکته (BUGS & NOTES)

نکته: برای ایجاد لینک سمبلیک، حتماً از -s استفاده کنید. در غیر این صورت، لینک سخت ایجاد می‌شود.

نکته: لینک‌های سخت نمی‌توانند برای پوشه‌ها ایجاد شوند و فقط برای فایل‌ها هستند.

نکته: اگر فایل اصلی را حذف کنید، لینک سمبلیک می‌شکند (و خطا می‌دهد) ولی لینک سخت همچنان معتبر است.

✍️ تمرین

تمرین ۱: یک فایل بسازید و با ln -s یک لینک سمبلیک به آن ایجاد کنید.

تمرین ۲: با ls -l لینک سمبلیک را بررسی کنید.

تمرین ۳: یک لینک سخت با ln (بدون -s) ایجاد کنید.

تمرین ۴: فایل اصلی را حذف کنید و ببینید لینک سمبلیک و سخت چه وضعیتی دارند.
