📦 دستور zip – فشرده‌سازی با فرمت ZIP

سطح: متوسط | زمان مطالعه: ۱۰ دقیقه

📖 نام (NAME)

zip – فشرده‌سازی فایل‌ها با فرمت ZIP (سازگار با ویندوز و مک)

💻 خلاصه (SYNOPSIS)

zip [گزینه] [فایل خروجی] [فایل‌های ورودی]

توضیح: zip یک ابزار فشرده‌سازی است که فایل‌ها را با فرمت ZIP (که در ویندوز و مک نیز قابل استفاده است) فشرده می‌کند. این دستور برای اشتراک‌گذاری فایل‌ها با کاربران ویندوز بسیار مفید است.

📖 توضیح کامل (DESCRIPTION)

دستور zip یک ابزار فشرده‌سازی است که فایل‌ها را به فرمت .zip تبدیل می‌کند. این فرمت در تمام سیستم‌عامل‌ها (ویندوز، مک، لینوکس) قابل استفاده است و برای اشتراک‌گذاری فایل‌ها بسیار رایج است.

zip می‌تواند فایل‌ها را با رمز عبور محافظت کند و همچنین می‌تواند فایل‌های جدید را به یک آرشیو موجود اضافه کند.

⚙️ لیست کامل گزینه‌ها (OPTIONS)

  • -r – فشرده‌سازی بازگشتی (برای پوشه‌ها و زیرپوشه‌ها).
  • -q – حالت بی‌صدا (quiet).
  • -v – نمایش جزئیات (verbose).
  • -d – حذف فایل از آرشیو.
  • -u – به‌روزرسانی فایل‌ها در آرشیو.
  • -e – رمزگذاری با رمز عبور.
  • -P – تنظیم رمز عبور (ناامن).
  • -T – تست صحت فایل‌های فشرده.
  • -x – استثنا کردن فایل‌ها (exclude).
  • -l – نمایش لیست فایل‌های موجود در آرشیو.
  • -o – تنظیم زمان آرشیو با آخرین فایل.
  • -z – اضافه کردن توضیح به آرشیو.
  • -0 – بدون فشرده‌سازی (ذخیره فقط).
  • -1 تا -9 – سطح فشرده‌سازی (1 = سریع‌ترین، 9 = بهترین).
  • --help – نمایش راهنمای کوتاه.
  • --version – نمایش نسخه‌ی دستور.

🧪 مثال‌ها (EXAMPLES)

مثال ۱: فشرده‌سازی چند فایل

zip archive.zip file1.txt file2.txt
خروجی: 
  adding: file1.txt (stored 0%)
  adding: file2.txt (stored 0%)

فایل‌های file1.txt و file2.txt را در archive.zip فشرده می‌کند.

مثال ۲: فشرده‌سازی یک پوشه (بازگشتی)

zip -r archive.zip myfolder/
خروجی: 
  adding: myfolder/ (stored 0%)
  adding: myfolder/file1.txt (stored 0%)
  adding: myfolder/file2.txt (stored 0%)

پوشه‌ی myfolder/ و تمام محتویات آن را فشرده می‌کند.

مثال ۳: فشرده‌سازی با رمز عبور

zip -e secure.zip secret.txt
خروجی: 
Enter password:
Verify password:
  adding: secret.txt (stored 0%)

secret.txt را با رمز عبور فشرده می‌کند. با -e از شما رمز عبور می‌خواهد.

مثال ۴: استخراج فایل ZIP (با unzip)

unzip archive.zip
خروجی: 
Archive:  archive.zip
  inflating: file1.txt
  inflating: file2.txt

archive.zip را در پوشه‌ی فعلی استخراج می‌کند. (دستور unzip برای استخراج استفاده می‌شود.)

مثال ۵: استخراج در یک پوشه خاص

unzip archive.zip -d /tmp/
خروجی: 
Archive:  archive.zip
  inflating: /tmp/file1.txt
  inflating: /tmp/file2.txt

archive.zip را در پوشه‌ی /tmp/ استخراج می‌کند.

مثال ۶: نمایش لیست فایل‌های ZIP

unzip -l archive.zip
خروجی: 
Archive:  archive.zip
  Length      Date    Time    Name
---------  ---------- -----   ----
        0  2024-07-20 10:00   file1.txt
        0  2024-07-20 10:01   file2.txt
---------                     -------
        0                     2 files

لیست فایل‌های موجود در archive.zip را نمایش می‌دهد.

🧠 نکته (BUGS & NOTES)

نکته: اگر zip روی سیستم شما نصب نیست، با sudo apt install zip unzip آن را نصب کنید.

نکته: برای استخراج فایل‌های .zip از دستور unzip استفاده کنید.

نکته: سطح فشرده‌سازی -9 بهترین فشرده‌سازی را ارائه می‌دهد، اما زمان بیشتری می‌برد.

✍️ تمرین

تمرین ۱: چند فایل بسازید و با zip آن‌ها را فشرده کنید.

تمرین ۲: یک پوشه را با zip -r فشرده کنید.

تمرین ۳: با unzip -l محتویات فایل .zip را ببینید.

تمرین ۴: فایل .zip را با unzip استخراج کنید.
